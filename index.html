How to Use

Add Node

Type your idea in the box and click 'Add Node' to create a new bubble.

Connect

Click a node to select it (it turns green), then add a new node to link them automatically.

Move

Drag and drop nodes anywhere on the canvas to organize your thoughts.

Edit/Delete

Double-click to text edit. Right-click to delete a node.
